Hoofdscherm


Het hoofdscherm blijft beschikbaar zolang RadixPro actief is. Vanaf dit scherm kun je alle functies van RadixPro bereiken. Voor de meeste functies geldt wel dat je eerst een horoscoop moet hebben berekend.

De knoppen

De belangrijske functies kun je via de volgende knoppen bereiken.

Nieuw: een horoscoop berekenen

Als je een horoscoop wilt berekenen klik je op de knop Nieuw. Je gaat dan automatisch naar het scherm Data invoer radix.

Figuur

De knop Figuur gebruik je om de horoscoop figuur te tonen. Je komt dan automatisch in het scherm Horoscoop Figuur.
Deze knop kun je pas gebruiken nadat je een horoscoop hebt berekend.

Posities

Met de knop Posities krijg je een overzicht van de berekende standen in een tabel. Je komt automatisch in het scherm Posities.
Deze knop kun je pas gebruiken nadat je een horoscoop hebt berekend.

Afsluiten

Met de knop Afsluiten sluit je RadixPro af. Alle schermen die nog geopend zijn worden ook afgesloten.

Menu

Boven in het scherm vind je een menu. Hiermee kun je de volgende acties uitvoeren:
  • Horoscoop
    • Nieuwe horoscoop. Als de knop Nieuw.
    • Afsluiten. Als de knop Afsluiten.
  • Resultaten
    • Horoscoop figuur. Als de knop Figuur.
    • Posities. Als de knop Posities.
  • Analyse
    • Aspecten. Een overzicht van aspecten met gemaakte orb. Je gaat naar het scherm Analyse :: aspecten
    • Halfsommen. Een overzicht van halfsommen met gemaakte orb. Je gaat naar het scherm Analyse :: halfsommen
  • Vookeuren
    • Configuratie. Opgeven vna je eigen configuratie.
    • Instellingen. Standaard warden voor tijdsoort en plaats.
  • Help
    • In houd. Inhoud van deze helppagina's.
    • Index. Begrippen uit deze helppagina's.
    • Over... Een korte toelichting bij RadixPro.
